# AGROBIASHARA Frontend

A complete, fully functional e-commerce frontend for farm produce marketplace.

## 📁 Project Structure

```
agrobiashara_frontend/
├── index.html              # Home/Landing page
├── css/
│   └── style.css          # All styling
├── js/
│   └── main.js            # All functionality
├── pages/
│   ├── products.html      # Products listing page
│   ├── cart.html          # Shopping cart
│   ├── checkout.html      # Checkout form
│   ├── order-confirmation.html  # Order confirmation
│   ├── login.html         # Login page
│   └── register.html      # Registration page
└── images/                # Image folder (create and add images here)
```

## 🖼️ Image Placement Guide

All image directives are marked with `[img src="..."]` comments in the code.

### Logo Image
- **Recommended size:** 60x60px
- **Format:** PNG or JPG
- **Location:** `images/logo.png`

### Hero Banner Image
- **Recommended size:** 1920x400px
- **Format:** JPG
- **Location:** `images/hero-banner.jpg`

### Product Images
- **Recommended size:** 200x200px each
- **Format:** JPG
- **Location:** `images/product-1.jpg` through `images/product-12.jpg`

## ✅ All Buttons Work

- **SHOP NOW** → Products page
- **Add to Cart** → Adds product with notification
- **Search** → Finds products or shows "Product not found"
- **Checkout** → Places order
- **Login/Register** → Creates user session

## 🚀 Quick Start

1. Extract the folder
2. Create `images` folder
3. Add your images
4. Open `index.html` in browser
5. Start shopping!
